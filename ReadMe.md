Sinatra Web Application Project for Flatiron School

This application is a simple social media clone, allowing the user to post, edit and delete posts and message other users.

Instructions:
1. Run bundle install in terminal
2. Run shotgun in terminal
3. Navigate to localhost:9393 in browser
